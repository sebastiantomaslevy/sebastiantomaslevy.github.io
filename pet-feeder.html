Automated Mechatronic Pet Feeder

Tech Stack: Arduino Uno, C++, NEMA 17 Stepper Motor, RTC Module

← Back to Portfolio

The Objective

Design and build a fully automated dry-food dispenser that delivers precise portions at user-scheduled intervals — solving the problem of irregular feeding times for pet owners.

System Architecture

Three main subsystems controlled by an Arduino Uno:

  • Actuation: NEMA 17 stepper motor driving a custom 3D-printed auger screw for consistent, measured food delivery.
  • User Interface: 16×2 LCD screen with buttons to set current time and configure the feeding schedule.
  • Sensing: DS3231 RTC module for accurate timekeeping, plus IR sensors to detect low food levels and alert the user.

Logic & Code

The system uses a state-machine architecture to handle user input without blocking the timing loop — ensuring the clock never misses a feed event while buttons are being pressed.

// Dispensing logic — runs every loop cycle if (currentHour == feedHour && currentMinute == feedMinute) { lcd.print("Dispensing..."); stepper.step(STEPS_PER_REVOLUTION * portions); logFeedingTime(); }

The Outcome

The final prototype successfully dispensed kibble with ±5% weight accuracy across repeated trials. The project received top marks in the Instrumentation course for clean wiring and robust error handling.
