Automated Mechatronic Pet Feeder

Course: Instrumentation | Tech Stack: Arduino, C++, Stepper Motors, LCD Display

The Objective

To design and build a fully automated dry-food dispenser that dispenses precise portions at user-scheduled intervals, solving the problem of irregular feeding times for pet owners.

System Architecture

I integrated three main subsystems controlled by an Arduino Uno:

  • Actuation: NEMA 17 Stepper motor with a custom 3D-printed auger screw for consistent food delivery.
  • User Interface: 16x2 LCD screen with buttons to set current time and feeding schedule.
  • Sensing: RTC (Real Time Clock) module for accurate timing and IR sensors to detect low food levels.

Logic & Code

The system uses a state-machine architecture to handle user input without blocking the timing loop.

// Snippet of the dispensing logic if (currentHour == feedHour && currentMinute == feedMinute) { lcd.print("Dispensing..."); stepper.step(STEPS_PER_REVOLUTION * portions); logFeedingTime(); }

The Outcome

The final prototype successfully dispensed kibble with ±5% weight accuracy. The project received top marks in the Instrumentation course for its clean wiring and robust error handling.
